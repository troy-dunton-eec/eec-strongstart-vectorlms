Using the catalog

Our catalog functions like an e-commerce shopping site:

  • Use the catalog filters to find courses.
  • Click the course title to read the description and objective.
  • When you're ready, click Get Item for the course you want to take.
  • To enroll in the course, click the Finish button. The course will be added to your Dashboard.
  • To start the course, click the Course Title and then select the Module you want to view.
